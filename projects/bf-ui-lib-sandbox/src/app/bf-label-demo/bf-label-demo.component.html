Applicable classes:

If you add a value to the label (bfValue), by default that falls under the label. If you like it right next to the label, use class "value-inline":

{{ valueInlineExample }}

To align a list of horizontal label - value keys, you can use class "value-list". It applies a flex display on the label / value. aligning them to the center.
It is also possible to adjust the flex of the value, to give more room to the right side using "value-list-2", "value-list-3", "value-list-4" and so on: 

{{valueListExample}}
