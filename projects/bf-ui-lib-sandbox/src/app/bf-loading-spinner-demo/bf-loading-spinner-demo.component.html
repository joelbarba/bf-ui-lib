This generates a component the necessary styles to display the spinner.

Applicable classes:

It is possible to change the size of the spinner with the classes:

  • {{sizeCss.sm}}
  • {{sizeCss.md}}
  • {{sizeCss.lg}}
  • {{sizeCss.xl}}

By default the spinner is an inline element. You can turn it centered to it's container applying the "center" class. By doing that though, the spinner becomes a block element.
It is also possible to center the spinner to one side only

  • {{centerCss.top}}
  • {{centerCss.right}}
  • {{centerCss.left}}
  • {{centerCss.bottom}}
