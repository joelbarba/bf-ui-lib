The component wraps the notification box element, to be displayed on the top right corner of the screen.
The you can push notifications through BfGrowlService service methods.
Every message is pushed and displayed for 2 seconds (by default). There is a vanishing animation that lasts .6s, so it total they are displayed 2.6s.
We can display multiple messages at the same time. The component handles a list of current messages, one above the other, although those cases should be rare.
The message text is automatically translated through the provided TranslateService of the application.
