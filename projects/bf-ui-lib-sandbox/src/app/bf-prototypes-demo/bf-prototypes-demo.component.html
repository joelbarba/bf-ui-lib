Array functions
Array.getById(id: string): T
Returns the first object (if any) in the array that matches by .id
Array.getByProp(property: string, value: any): T
Returns the first object (if any) in the array that matches the given property / value
Array.getIndexById(id: string): number
Returns the index of the first object (if any) in the array that matches by .id. If not -1
Array.getLast(): T
Returns the last element of the array, or undefined if it's empty
Array.removeById(id: string): T
Removes the first object (if any) in the array that matches by .id. Returns the removed object or null if not found.
Array.removeByProp(property: string, value: any): T
Removes the first object (if any) in the array that matches by property / value. Returns the removed object or null if not found.
Array.dCopy(): Array{{'<' + 'T' + '>'}}
Array clone - Makes an exact (deep) copy of the array, with no references, and returns it. Every item inside is deeply copied too.
Object functions
Object.keyMap(propNames: string): Partial{{'<' + 'T' + '>'}}
It returns the same object but only with the selected properties. Ex: user.keyMap('id, name');
Object.cloneProp(propName: string, fromObj: Object): T
Clones a property from an other object to itself. If that value is null / undefined, or not found on the other object, it deletes the property of the object. So after that both objects have exactly the same (property presence & value)
Object.dCopy(): T
Object clone - Makes an exact (deep) copy of the object, with no references, and returns it. Every item inside is deeply copied too.
Number functions
Number.isIn(list: Array{{'<' + 'number' + '>'}}): boolean
Returns true if there is any instance of the current number into the list
String functions
String.isIn(list: Array{{'<' + 'string' + '>'}}): boolean
Returns true if there is any instance of the current string into the list
String.pad(leftPad: number, placeholder = '0', rightPad?: number): string
It fulfills the string with 0s (or specified placeholder) on the left & right.
Shortcut for .padStart() / .padEnd(). Ex: myValue.pad(4) -> 0003
Date functions
Date.toUTC()
Converts the time, whatever its timezone is, to its UTC equivalent. It preserves the same JS timezone
Date.truncMin()
Truncate the current date to minutes
Date.truncHour()
Truncate the current date to hours
Date.truncDay()
Truncate the current date to days
Date.addSeconds(seconds: numbers)
Add seconds to the current date
Date.addMinutes(minutes: numbers)
Add minutes to the current date
Date.addHours(hours: numbers)
Add hours to the current date
Date.addDays(days: numbers)
Add days to the current date
Date.addMonths(months: numbers)
Add months to the current date
Date.addYears(years: numbers)
Add years to the current date
Date.convertTZ(desTZ?: string)

Convert a JS time from your current timezone to another timezone (Timezone standard list).
The object returned is a JS date with the time at the specified timezone, still represented in the current timezone. Example: 

        let myDate = new Date('Mon Jul 29 2019 12:00:55 GMT+0100 (Irish Standard Time)')
        myDate.convertTZ('America/New_York');
        console.log(myDate); ---> Mon Jul 29 2019 07:00:55 GMT+0100 (Irish Standard Time)

Import

An index js module is provided with an init function BfPrototypes.
That function can be called to extend all listed prototypes (if no parameters), or can be also called passing
the list of functions that we want to extend. That list is composed by strings that contain 2 values (type and function name) separated by a '.'
