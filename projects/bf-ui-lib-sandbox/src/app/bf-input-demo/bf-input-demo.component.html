External Control Example (extCtrl$):

It is possible to control some options of the input externally. There are 2 ways to do it: With "extCtrl$" or hooking up (bfOnLoad) event

You can pass an observable in "extCtrl$". The component will subscribe and listen to it, so actions can be triggered from outside. You can use a Subject if you need to trigger them manually. The .next() should send an object with an "action" and "value"

  • {{ctrlAction}}
 
{{extCtrlExample}}

It is also possible to use the returned object in (bfOnLoaded) to trigger internal actions.
You should make sure though that the object has been initialized before using it. That's done on ngAfterViewInit lifecycle.
