This component is to be used when having a list with multi selection. It works using a BfListSelection object, and there are 2 use cases of the component:

  • With [id]: A checkbox for every item on the list, to select/unselect it.
  • Without [id]: A checkbox to select/unselect all the items of the list (current page).
The BfListSelection exposes an object (.ids) with the list of selected items on the list. You can use the method .getSelection() to retrieve the list of id into an array.
