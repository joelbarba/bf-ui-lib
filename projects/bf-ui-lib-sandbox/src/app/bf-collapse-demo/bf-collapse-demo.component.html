Collapsible wrapper for dynamic content. It applies a transitional animation. Notice that this component is applied as an attribute: [bf-collapse]
It exposes the status on the view template.

This component uses angular animations. In order to make it work you need to import BrowserAnimationsModule into your module

The component is exported on the "bfCollapse" view element (#ref="bfCollapse), so the internal methods can also be used manually
